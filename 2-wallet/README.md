# Solana 钱包生成工具

提供多种方法生成和管理 Solana 钱包，包括通过助记词生成、通过私钥生成、HD 钱包派生，以及创建具有自定义前缀/后缀的"靓号"钱包。

## 功能特点

- **多种钱包生成方式**：
  - 随机生成新钱包和助记词
  - 从私钥（Base58 格式）生成钱包
  - 从单个助记词派生多个钱包（HD 钱包）
  - 生成具有特定前缀/后缀的"靓号"钱包

- **多线程支持**：
  - 使用 Node.js 工作线程加速"靓号"钱包的生成
  - 自动使用系统可用 CPU 核心
  - 支持搜索进度跟踪和超时设置

- **完整钱包信息展示**：
  - 显示地址
  - 显示 Base58 编码的私钥
  - 显示 Unit8Array 编码的私钥
